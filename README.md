# Kasparro AI Agentic Content Generation System

This is a modular, multi-agent system designed to generate structured web content from raw product data using Google Gemini.

## How to Run
1. Clone the repo.
2. Install requirements: `pip install -r requirements.txt`
3. Run the app: `streamlit run app.py`
